Here are all the SVG components that make up the scene, broken out as standalone pieces.

## Shared gradient/pattern defs

Every scene reuses these. Paste the `<defs>` block once at the top of your SVG, then reference the IDs below.

```svg
<defs>
  <linearGradient id="nightSky" x1="0" y1="0" x2="0" y2="1">
    <stop offset="0%" stop-color="#0a1540"/>
    <stop offset="40%" stop-color="#1a3060"/>
    <stop offset="70%" stop-color="#2a6878"/>
    <stop offset="100%" stop-color="#3a9880"/>
  </linearGradient>
  
  <linearGradient id="auroraGlow" x1="0" y1="0" x2="0" y2="1">
    <stop offset="0%" stop-color="#6ad4a0" stop-opacity="0"/>
    <stop offset="50%" stop-color="#6ad4a0" stop-opacity="0.5"/>
    <stop offset="100%" stop-color="#4ab8a8" stop-opacity="0.7"/>
  </linearGradient>
  
  <radialGradient id="moonGlow" cx="0.5" cy="0.5" r="0.5">
    <stop offset="0%" stop-color="#ffffff" stop-opacity="0.9"/>
    <stop offset="40%" stop-color="#e8f4d8" stop-opacity="0.4"/>
    <stop offset="100%" stop-color="#a8e8c4" stop-opacity="0"/>
  </radialGradient>
  
  <radialGradient id="moon" cx="0.4" cy="0.4" r="0.6">
    <stop offset="0%" stop-color="#ffffff"/>
    <stop offset="70%" stop-color="#f0f8e0"/>
    <stop offset="100%" stop-color="#c8e0b8"/>
  </radialGradient>
  
  <linearGradient id="mountain" x1="0" y1="0" x2="0" y2="1">
    <stop offset="0%" stop-color="#3a5068"/>
    <stop offset="100%" stop-color="#2a3a50"/>
  </linearGradient>
  
  <linearGradient id="meadowNight" x1="0" y1="0" x2="0" y2="1">
    <stop offset="0%" stop-color="#1a3a4a"/>
    <stop offset="100%" stop-color="#0a2030"/>
  </linearGradient>
  
  <radialGradient id="purpleGlow" cx="0.5" cy="0.5" r="0.5">
    <stop offset="0%" stop-color="#d4a8ff" stop-opacity="0.85"/>
    <stop offset="100%" stop-color="#8a5ac4" stop-opacity="0"/>
  </radialGradient>
</defs>
```

## Layer order

Stack from background to foreground when assembling — later elements render on top:

Sky background → aurora gradient overlay → stars → moon → upper-right clouds → dark swooping cloud → mid-sky wispy cloud → horizon aurora band → mountains → water strip → meadow foreground → fireflies (behind flowers) → flowers (largest/closest last so they sit on top).

## Color palette

In case you want to retint anything — every hex used in the scene:

**Sky:** `#0a1540` → `#1a3060` → `#2a6878` → `#3a9880`
**Aurora:** `#6ad4a0`, `#4ab8a8`, `#7ae8b0`, `#a8f0c4`
**Moon:** `#ffffff`, `#f0f8e0`, `#c8e0b8`, `#e8f4d8`, `#a8e8c4`, `#c8d8b8` (craters)
**Clouds:** `#4a7a98` / `#6b98b8` (light), `#1a2540` / `#8bc4d4` (swooping), `#3a6878` (wispy)
**Mountains:** `#3a5068` → `#2a3a50`
**Water:** `#2a5868`, highlights `#6ad4a0` / `#8be8c4`
**Meadow:** `#1a3a4a` → `#0a2030`
**Flower palette:** `#3a1a7a` (calyx), `#4a2a9a`, `#5a2ab4` (back), `#7a3ad4` (sides), `#9a5af0` (front), `#2a0a5a` (center), `#e8c8ff` (sparkle), `#d4a8ff` → `#8a5ac4` (glow), `#2a4a3a` (stem)

Each piece is independent so you can remix — drop the moon into a different scene, swap the flowers for a different species, etc.
